# 📊 Student Grade Calculator (حاسبة التقديرات)

تطبيق بسيط بواجهة مستخدم رسومية (GUI) لحساب مجموع درجات الطلاب ونسبتهم المئوية وتحديد التقدير العام.

## ✨ المميزات:
- واجهة مستخدم بسيطة ومريحة باستخدام مكتبة **Tkinter**.
- حساب تلقائي للمجموع والنسبة المئوية من 300.
- فرز التقديرات بدقة (امتياز، جيد جداً، جيد، مقبول، راسب).
- معالجة الأخطاء في حال إدخال نصوص بدلاً من أرقام.
- زر مدمج للانتقال إلى حساب المطور مباشرة.
