# 🎓 نظام حساب المعدلات الذكي (Smart Grade & GPA Calculator)

تطبيق احترافي مطور بواجهة مستخدم رسومية (GUI) حديثة لحساب التقديرات الأكاديمية والمعدل التراكمي (GPA) لـ 5 مواد دراسية.

## ✨ المميزات المضافة في التحديث:
- **واجهة ملوّنة وحديثة:** تصميم جذاب بخلفية داكنة متناسقة مريحة للعين.
- **توسيع المواد:** دعم إدخال درجات 5 مواد دراسية بدلاً من 3.
- **حساب الـ GPA:** حساب تلقائي للمعدل التراكمي من (4.0) بدقة.
- **منظومة التحقق:** فحص تلقائي لمنع إدخال درجات خارج النطاق المنطقي (من 0 إلى 100).
- **معالجة الأخطاء الذكية:** حماية البرنامج من الانهيار عند إدخال نصوص أو خانات فارغة.
- **ربط مباشر:** زر مدمج ينقل المستخدم بضغطة واحدة إلى مستودع المشروع على GitHub.
